# Бот для автоматизации торговли на Bybit

- опрашивает раз в установленный промежуток времени биржу
- размещает сетку ордеров на покупку
- при изменении состояния ордеров на покупку создает / сдвигает продажный ордер так, чтобы общая прибыль соответствовала установленной

# TO START

- разместите .env файл в корне проекта, заполнив его на основе .env.example
- выполните docker compose up
